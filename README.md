A challenging and fun 2D puzzle game built with Unity. Navigate through increasingly complex levels by solving puzzles and overcoming obstacles with your rocket!

## 🎮 About

Rocket Boost is an engaging 2D puzzle game developed in Unity where players must guide their rocket through various challenging scenarios. Each level presents new puzzles to solve and mechanics to master, providing a progressively rewarding gameplay experience.

## ✨ Features

- **Engaging Puzzle Mechanics** - Thoughtfully designed puzzles that challenge your problem-solving skills
- **Progressive Difficulty** - Start easy and work your way up to increasingly complex challenges
- **Beautiful 2D Graphics** - Clean and colorful visual design
- **Smooth Controls** - Responsive and intuitive gameplay
- **Multiple Levels** - Numerous levels to complete and master

## 🛠️ Getting Started

### Prerequisites

- **Unity** 2020 LTS or later (or check your project's Unity version)
- **Git** for version control

### Installation

1. Clone the repository:
```bash
git clone https://github.com/bartusanli/Rocket-Boost.git
```

2. Open the project in Unity:
   - Launch Unity Hub
   - Click "Open" and select the cloned project folder
   - Wait for Unity to import the project assets

3. Open the main scene and press Play to test

## 🎯 How to Play

- Use A,D to rotate and SPACE to fly your rocket
- Solve each puzzle by navigating through obstacles
- Reach the goal to advance to the next level
- Master all levels to complete the game!

*(Update these controls based on your actual game mechanics)*
